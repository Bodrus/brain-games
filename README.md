# Brain Game

## Installation Guide

* npm install -g project-lvl1-s216

### The project contains six console games.
### To start the game you need to enter one of them in the console:
* brain-balance 
* brain-calc
* brain-even
* brain-gcd
* brain-prime
* brain-progression
